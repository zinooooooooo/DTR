# OJT DTR Manager (Web)

Modern and responsive Daily Time Record (DTR) Management Web App for OJT students.

## Features
- Official schedule setup (start/end time)
- Time In / Time Out logging
- Auto-calculation of total hours per day
- Late & overtime detection (with color indicators)
- Daily records table (search, edit, delete)
- Required OJT hours tracker + progress bar
- Dashboard summary (days worked, total hours, total overtime)
- LocalStorage persistence
- Export records to CSV
- Dark mode toggle

## Run locally
From this folder:

```bash
npm install
npm run dev
```

Then open the dev server URL printed in your terminal.
